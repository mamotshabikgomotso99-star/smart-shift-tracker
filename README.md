# Smart Shift & Task Tracker

A simple web app to track work shifts and tasks in real-time.  
This project helps you **stay organized**, **track your hours**, and **manage tasks efficiently**, making it perfect for work or personal productivity.

---

## Features
- Start and end work shifts with a live timer
- Add tasks during a shift with timestamps
- Auto-generate a clean shift summary
- Persistent data using `localStorage`
- Lightweight, fast, and fully client-side (no backend required)

---

## Tech Used
- **HTML** – Structure and layout  
- **CSS** – Styling for a clean UI  
- **JavaScript** – Logic for timer, tasks, and summary  
- **localStorage** – Saves data between sessions
